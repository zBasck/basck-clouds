# Contribuindo com o Basck Clouds

Obrigado por considerar contribuir! 🎉

## Como rodar localmente
1. `git clone https://github.com/zBasck/basck-clouds.git`
2. `cd basck-clouds`
3. `npm install`
4. `npm run dev`

## Padrões
- TypeScript estrito
- Prettier (formatação automática)
- Commits semânticos (`feat:`, `fix:`, `docs:`, `chore:`)

## Pull requests
- Crie uma branch a partir de `main`
- Faça commits pequenos e descritivos
- Garanta que `npm run build` passa antes de abrir o PR
